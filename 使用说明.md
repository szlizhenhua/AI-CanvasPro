# AI 画布《使用说明》

本文面向“普通使用者”，目标是让你不看代码也能把当前版本的全部功能用起来（包括入口位置、操作步骤、快捷键、常见问题）。

> 提示：本项目是“节点 + 连线”的工作流。上游节点输出会流向下游节点，提示词里也可以用 `@` 直接引用节点结果。

***

## 1. 快速上手（3 分钟）

### 1.1 启动

- 一键整合包：双击“双击运行.bat”，浏览器打开 `http://localhost:8777`
- 源码运行：按项目根目录 README 里的“快速开始”操作

### 1.2 配置 API Key（第一次使用必做）

1. 点击左下角头像（圆形 “A”）→ 选择“设置”
2. 进入“API 输入”
3. 填写对应平台的 Key（RunningHub / APImart / PPIO / GRSAI / OpenAI 兼容）
4. 点击“保存”

### 1.3 生成一张图（最典型流程）

1. 画布左侧点击“+”（添加菜单）
2. 选择“图像”（创建 AI 图像生成节点）
3. 在节点下方的提示词输入框里输入描述
4. 如需参考图：把一个“源图像节点 / 生成图像节点”连到它的输入上（见“3. 节点与连线”）
5. 点击节点内的“生成”按钮

生成结果会以**本地文件**形式保存到电脑里，并在画布中显示。

## 2. 界面总览（你会看到什么）

### 2.1 顶部栏

- 底部头像菜单：设置 / 使用教程（入口可能为占位）/ 关于

### 2.3 左下角画布控制

- 小地图开关（按钮提示 `M`）
- 网格吸附开关（按钮提示 `L`）
- 适应画布（按钮提示 F ）
- 缩放滑杆

## 3. 画布基础操作（缩放/平移/选择/移动/连线）

### 3.1 平移与缩放

- 鼠标滚轮：缩放画布
- 按住空格不放 + 鼠标拖动：平移画布
- 缩放滑杆：调整缩放比例
- “适应画布”：把视图对齐到当前节点整体范围

### 3.2 选择与多选

- 单击节点：选中
- Shift + 单击：多选（可在“快捷键设置”里调整多选修饰键）
- 框选：在空白处按下并拖动（框住多个节点）
- Ctrl + A：全选

### 3.3 移动/复制/撤销/删除

- 拖拽节点：移动
- Ctrl + C / Ctrl + V：复制/粘贴节点
- Ctrl + Z / Shift + Ctrl + Z：撤销/重做
- 删除节点：默认是 `D`（可在“快捷键设置”里改成 Delete/Backspace）
- Alt + 拖拽：复制节点并立刻进入拖拽（会把选中的节点与它们之间的连线一起复制）

### 3.4 连线（把上游结果接给下游）

典型用法：把“参考图 / 参考音频 / 参考文本 / 参考视频”接到 AI 生成节点上，或把某个生成结果继续喂给下游节点。

- 方法 A（常规）：鼠标悬停到节点边缘，出现连接锚点 → 从输出锚点拖到目标节点输入锚点
- 方法 B（拾取连线）：在部分 AI 节点的引用条左侧有“指针/吸附”小按钮，点击后进入拾取模式 → 再点击一个节点完成连接（Esc 可退出）

## 4. 节点类型与用法

节点就是画布里的“功能块”。你把节点连起来，就像搭积木一样把流程串起来。

### 4.1 源节点（导入/承载素材）

源节点用来“放素材”，属于上游输入：

- 源文本：放一段文本（可被下游引用）
- 源图像：放图片（可作为参考图/素材图）
- 源视频：放视频（可用于裁剪、抠像、场景检测等）
- 源音频：放音频（可用于裁剪、作为参考或素材）

创建方式：

- 左侧“添加菜单”→ “添加资源”→ “上传”
- 直接把文件拖拽到画布（图片/视频/音频等会自动上传并在落点创建对应 source 节点）

### 4.2 AI 生成节点（文本/图像/视频/音频）

AI 生成节点用来“产出内容”，属于核心生产节点：

- 生成文本：写文案、改写、总结、脚本等
- 生成图像：文生图/参考图生图等
- 生成视频：文生视频/图生视频等
- 生成音频：文字转语音等

通用操作（所有 AI 节点通用的“输入体系”）：

- 连线引用：把上游节点连到当前节点，当前节点会在“引用条”里显示缩略预览
- `@` 引用：在提示词输入框里输入 `@`，会弹出引用菜单，选择任意节点，把它的输出嵌入提示词
- `/` 预设：在提示词输入框里输入 `/`，弹出“预设生成”菜单（不同节点类型有不同预设）

### 4.3 分镜节点（Storyboard）

用途：把多张图按网格排成分镜，并一键“合成”生成新的宫格图片结果。

入口：创建 `storyboard` 节点（当前版本一般通过工作流/菜单入口或导入项目出现）

节点内工具条：

- “比例”：切换分镜比例
- “网格”：切换 N×M
- “编辑”：进入编辑模式（允许在格子内部拖动图像/在格子间拖拽）
- “合成”：一键合成并导出为新的图片（会生成新的节点/文件）
- “清空”：清空所有格子
- “折叠/展开”：收起展示

### 4.4 场景检测节点（Scene Detection）

用途：对视频做“场景切换点检测”，生成切分建议，并支持“一键裁剪/导出场景”。

使用步骤：

1. 创建“场景检测”节点
2. 把一个视频节点连到它的输入（节点里会显示“视频源”已接入）
3. 调整“检测敏感度”滑杆
4. 点击“开始检测”
5. 检测完成后可按列表对单段场景点击“裁剪”，或点击“自动裁剪”“导出场景”

### 4.5 组节点（Group）

用途：把多个节点打包成组，便于整体移动与管理。

- 选择多个节点 → Ctrl + G 打组

## 5. 节点内工具（图片/视频/文本/音频）

说明：这些工具入口来自“节点浮动工具栏”（选中节点或鼠标移到节点上时出现）。

### 5.1 图片节点工具

常见按钮：

- 遮罩编辑器：进入“抠图/遮罩”编辑
- 高清：对当前图执行高清修复/放大（具体依赖配置的模型/提供商）
- 扩图：进入扩图模式（可调比例/尺寸/模型）
- 宫格裁剪：把图片按网格切成多张图并生成节点
- 控制角度：多角度生成入口（具体行为以当前版本为准）
- 标注：进入标注模式（画笔/矩形/橡皮擦/颜色/撤回重绘/保存）
- 裁剪：进入裁剪模式（可选比例，确认后生成新图）
- 全屏显示：全屏预览
- 下载：下载当前图片

#### 5.1.1 标注（Annotate）

入口：图片节点 → 点“标注”

可用操作：

- 工具：画笔 / 矩形 / 橡皮擦
- 颜色：红/橙/黄/绿/蓝/紫/白
- 撤回/重绘/清除
- 保存（保存后会把结果写成新的图片文件，并更新节点显示）

#### 5.1.2 裁剪（Crop）

入口：图片节点 → 点“裁剪”

使用步骤：

1. 进入裁剪后，画面会压暗，出现裁剪框与底部工具条
2. 拖动裁剪框或拖动八个方向手柄调整范围
3. 点“比例”选择裁剪比例（原图/21:9/16:9/9:16/4:3/3:4/1:1）
4. 点“确认裁剪”生成裁剪结果
5. 按 `Esc` 退出不保存

#### 5.1.3 扩图（Expand / Outpaint）

入口：图片节点 → 点“扩图”

使用步骤：

1. 进入扩图后，会出现一个外框（扩图画布）和底部工具条
2. 在工具条选择：
   - 比例（原图/21:9/16:9/9:16/4:3/3:4/1:1）
   - 尺寸（1K/2K/4K）
   - 模型（下拉选择不同提供商/模型）
3. 调整扩图框位置/大小（比例锁定时只允许按比例调整）
4. 点击“生成扩图”
5. 按 `Esc` 退出

#### 5.1.4 遮罩编辑器（Matting）

入口：图片节点 → 点“遮罩编辑器”

当前可用操作（以工具条 tooltip 为准）：

- 画笔：`B`（再次按下可切换模式）
- 橡皮擦：`E`
- 油漆桶：`G`
- 撤回：Ctrl + Z
- 重绘：Ctrl + Y
- 清除打点：`R`
- 保存：保存遮罩结果（会写入节点并生成新的本地文件）
- 取消：Esc

说明：自动抠图入口在当前版本标记为“暂不可用/开发中”。

### 5.2 视频节点工具

常见按钮：

- 裁剪视频：进入视频裁剪条（选择区间 → 完成后生成新视频）
- 抠像：进入视频抠像（对当前帧打点 → 提交任务 → 生成抠像结果视频）
- 高清 / 全屏显示 / 下载

#### 5.2.1 裁剪视频（Video Clip）

入口：视频节点 → 点“裁剪视频”

使用步骤：

1. 进入后，视频节点下方会出现裁剪条（左右手柄 + 选区）
2. 拖动左右手柄选择要保留的时间区间
3. 点击“完成”（或按提示键位完成）
4. 会调用后端做物理切割，生成一个新视频文件，并创建新的视频节点
5. 按 `Esc` 取消

#### 5.2.2 视频抠像（Video Keying）

入口：视频节点 → 点“抠像”

打点规则（界面会提示）：

- 左键：选中目标（前景点）
- 右键：排除目标（背景点）
- `R`：清空所有点
- “设置”：可调分辨率、帧率、抠像模式（Sec/Sam3/MA2）、显存档位（24G/48G）
- 点“抠像”提交任务；如果任务正在运行，再点一次会取消该节点的抠像任务

### 5.3 音频节点工具

源音频节点通常提供：

- 上传音频
- 播放/暂停
- 裁剪音频
- 倍速
- 下载

其中“裁剪音频”会进入类似视频裁剪条的 UI，并在确认后通过后端生成新音频文件与新节点。

### 5.4 文本节点工具

常见按钮：

- 复制：复制文本内容
- 全屏显示：更舒适地查看长文本

## 6. 项目管理（保存/导入/导出/多画布）

### 6.1 保存

- 快捷键：Ctrl + S
- 或 Tab 右键菜单：保存

保存会把当前画布状态落到本地项目目录（整合包默认在 `user/Canvas Project/` 下）。

### 6.2 导入（打开别人分享的工作流）

- 直接把 `.json` 文件拖到页面（任意位置松手）
- 系统会自动创建/切换到同名 Tab 并加载

### 6.3 导出（另存为 JSON）

- 在 Tab 上打开菜单 → “另存为...”
- 会直接下载一个 `.json`

## 7. 资产面板（素材库）

用途：把画布上的节点集合保存成“资产”，以后可一键拖回画布复用（并保留资产内部的连线关系）。

### 7.1 创建资产

1. 在画布上选中一个或多个节点（建议多选相关节点）
2. 打开资产面板，点击“创建资产”（会弹出创建面板）
3. 填写“资产名称”，选择分类（人物/场景/物品/自定义）
4. 点击“创建”

资产创建会把你选中的节点，以及它们之间的连线一起保存起来。

### 7.2 使用资产

- 在资产面板里选择一个资产 → 点击“添加整个资产”（会在画布中心附近生成一套节点）
- 如果资产里有多个“子资产/条目”，也可以逐个添加

## 8. 快捷键（默认值 + 如何修改）

### 8.1 如何修改快捷键

1. 头像菜单 → 设置
2. 进入“键盘快捷键”
3. 点击某个动作的快捷键区域 → 按下你想设置的组合键
4. 保存（会同步到本地服务端配置）

### 8.2 默认快捷键清单（当前版本）

缩放与导航：

- 放大：Ctrl + `+`
- 缩小：Ctrl + `-`
- 聚焦/适应画布：`F`
- 小地图：`M`

编辑与选择：

- 复制节点：Ctrl + C
- 粘贴节点：Ctrl + V
- 撤销：Ctrl + Z
- 重做：Shift + Ctrl + Z
- 删除节点：`D`
- 全选：Ctrl + A
- 多选修饰：Shift
- 打组：Ctrl + G
- 网格吸附：`L`
- 剪刀（切断连线）：按住 Ctrl 进入切割模式（鼠标划过连线切断）

创建节点：

- 创建源文本：`T`
- 创建注释节点：`N`
- 创建生成文本：`Q`
- 创建生成图像：`W`
- 创建生成视频：`E`
- 创建生成音频：`R`

通用：

- 保存画布：Ctrl + S
- 撤回：Ctrl + Z
- 画笔（切换模式）：`B`
- 矩形：未设置
- 橡皮擦：`E`
- 油漆桶：`G`（遮罩编辑器可用）
- 清空：`R`
- 鼠标中键按住拖动：临时擦除，松开恢复原工具
- 鼠标右键按住拖动：临时擦除，松开恢复原工具

特殊模式按键拦截：

- 在“视频裁剪/视频抠像”界面里，为了避免误操作，很多快捷键会暂时失效；通常只保留 `Esc` 用来退出/取消。

## 9. 文件落盘与路径（你生成的东西在哪）

- 画布工程（JSON）：保存到本地项目目录（整合包默认 `user/Canvas Project/`）
- “节点生成”的媒体产物（图片/视频/音频等）：统一保存在 `output/` 目录，并以本地文件作为最终显示来源
- 用户手动上传的源文件：通常在 `data/uploads/`（具体由后端上传逻辑决定）
